# Quiz Game

A fun and interactive quiz game with a cute pink theme! Test your knowledge and enjoy the game.

## Features
- 5 General Knowledge Questions
- Score Tracker
- Cute Pink Theme

## How to Play
- Open `index.html` in your browser.
- Answer the questions to score points.
